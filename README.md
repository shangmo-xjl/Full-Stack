# Full Stack Development 学习项目

这是一个全栈开发学习项目，包含多个阶段(part)，记录了从基础到进阶的全栈开发学习过程。

## 项目结构

- **part0**: 基础示例应用 (example_app)
  - 使用 Node.js 构建的后端应用
  - 包含基本的服务器设置和路由处理
  - 使用 body-parser 等基本依赖

- **part1**: React + Vite 前端应用
  - 使用 React 19 和 Vite 构建的现代前端应用
  - 集成了 ESLint 进行代码检查
  - 支持开发环境热更新(HMR)

## 技术栈

- **前端**: React, Vite, JavaScript
- **后端**: Node.js
- **工具**: ESLint

## 学习目标

本项目旨在学习和实践全栈开发技能，包括前端框架使用、后端服务构建、前后端交互等技能。
